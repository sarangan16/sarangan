My Portfolio
Welcome to my Portfolio website, a personal showcase of my skills, projects, and professional experience. This interactive, smooth, and responsive portfolio is built using React, Tailwind CSS, and GSAP animations. With features like i18n for multi-language support and EmailJS for contact functionality, this portfolio serves as an interactive and dynamic introduction to who I am and what I do.

Features
Interactive UI: Smooth transitions and animations powered by GSAP, ensuring a visually engaging experience.

Responsive Design: Built with Tailwind CSS, ensuring it works seamlessly on both desktop and mobile devices.

Multi-language Support: The website is fully translated with i18n, allowing users to switch between English and German for a personalized experience.

Contact Me: Integrated EmailJS to provide an easy way for visitors to get in touch with me directly from the portfolio.

Project Showcase: Displaying a collection of my latest projects with a clean and elegant layout.

Introduction: A brief and concise introduction about me, including my skills and experience.

Technologies Used
React: For building the user interface with reusable components.

Tailwind CSS: For quick and efficient styling, making the website responsive and mobile-first.

GSAP: For smooth animations and transitions, adding a dynamic touch to the site.

i18n: For providing multi-language support with seamless translation (English & German).

EmailJS: For sending contact messages directly from the website without a back-end server.
